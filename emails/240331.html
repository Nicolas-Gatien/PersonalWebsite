Cellular Automata Life Explorer

I find the concept of life in cellular automata beautiful. Both from Conway's Game of Life to Lenia:

Both of these simulations are versions of cellular automata. A cellular automata is a grid of pixels, where the state of each pixel is determined by a set of rules.

In the Game of Life, the rules are:
1. If a cell is dead, it becomes alive if it has exactly 3 living neighbours.
2. If a cell is alive, it dies of isolation if it has 0 or 1 living neighbours.
3. If a cell is alive, it dies of overpopulation if it has 4 or more living neighbours.

Lenia on the other hand doesn't have rules per say. It is a convolutional cellular automata. Instead of having rules, it applies a kernel to the entire board on every timestep.

In both cases, it leads to interesting life forms emerging. Over the last week I've been working on a system that explores these environments to try and find settings that lead to the creation of life forms.
